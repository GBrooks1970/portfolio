# Test Automation Portfolio — landing page

A static, self-contained landing page for Gary Brooks's test-automation portfolio, published via
GitHub Pages: **https://gbrooks1970.github.io/portfolio/**

It links the seven portfolio projects (E2E web UI, API/BDD, multi-stack parity, real-time
WebSocket, a Screenplay teaching library, and a shipped product) with their repositories, live
demos, and CI status.

`index.html` is the whole site — no build step, no dependencies. Edit and push to `main`; Pages
redeploys automatically.
